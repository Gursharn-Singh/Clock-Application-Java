# ⏰ Clock Application (Java Swing)

A desktop-based clock application built using Java Swing.

## 🚀 Features
- Digital Clock (Real-time)
- Alarm System (HH:mm format)
- Stopwatch (Start, Stop, Reset)

## 🛠️ Tech Stack
- Java (JDK 17)
- Java Swing
- AWT

## 📸 How to Run
1. Compile:
   javac Project.java

2. Run:
   java Project

## 💡 Description
This application provides a user-friendly interface for time tracking, alarms, and stopwatch functionality.
